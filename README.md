# 📊 Retail Business Dashboard

## Overview

The Madhav Retail Business Dashboard is a project that helps us look at how a retail business is doing. This project uses Power BI to turn sales numbers into information. We can use this information to understand how money the business is making, if it is making a profit what customers are buying and how different areas are performing. The Madhav Retail Business Dashboard is very good at analyzing and monitoring the performance of a business.

## Purpose

The reason we made the Madhav Retail Business Dashboard is to help people make good decisions based on the numbers. It puts all the business numbers in one place so we can see what is happening and find patterns that can help the business do better.

## Key Features

- We have a dashboard for looking at retail sales
- We can see the most important numbers like

Total Sales
Total Profit
Quantity Sold
Average Order Value (AOV)
- We can look at sales by state to see which areas are doing well
- We can look at how different categoriesand sub-categories are doing
- We can see how profit changes from month to month
- We can look at sales by customer
- We can see how people are paying for things
- We have filters and slicers to help us look at the numbers in more detail
- We have many different types of charts like

Bar Charts
Pie Charts
Donut Charts
Clustered Bar Charts
Line Charts
Area Charts
Maps
- The layout is clean and easy to use so we can get the information we need

## Tools and Technologies Used
- We used Microsoft Power BI
- We used Power Query
- We used DAX (Data Analysis Expressions)
- We used Data Modeling
- We used Data Visualization

## Business Insights

- We found out which states and customers are doing the best
- We looked at how profitable different categories and sub-categories are
- We tracked how profit changes from month, to month and saw some patterns
- We looked at how customers like to pay for things
- We made it easy for people to make good decisions with our interactive charts and the Madhav Retail Business Dashboard
